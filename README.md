# Sales & Revenue Dashboard (Power BI)
Built an interactive Sales &amp; Revenue Dashboard using Power BI to analyze performance across time, product, and region. The dashboard features KPIs, dynamic visuals, slicers, and drill-throughs, enabling users to track trends and explore sales data from summary to transaction-level insights.

##  Business Objective
To analyze overall sales, revenue, profit, and quantity performance across different business dimensions and track monthly trends to support data-driven decision making.

## Tools & Technologies
- Power BI (DAX, Power Query, Data Modeling)
- SQL
- Excel / CSV Dataset

## Key KPIs
- Total Sales
- Total Revenue
- Total Profit
- Total Quantity
- Profit Margin %

## Dashboard Features
- KPI cards to highlight key business metrics
- Category-wise Sales and Profit analysis using bar charts
- Region-wise Revenue analysis using column charts
- Segment-wise Sales contribution using donut charts
- Monthly Sales trend using line charts
- Interactive slicers for Date, Region, and Category

## Drillthrough & Interactivity
- Drillthrough functionality to navigate from summary-level insights to detailed transactional data
- Dynamic filtering using slicers for flexible analysis


## Data Overview
- Sales transaction data including order date, category, region, sales, revenue, profit, and quantity

## File Structure
- Sales_revenue.sql : SQL queries for analyzing overall sales, revenue, profit, and quantity performance across different business dimensions and track monthly trends to support data-driven decision making.
- Sales and Revenue Dashboard.pbix : Power BI dashboard that visualizes the sql results and provides user friendly interface for exploring the data.


## Outcome
Enabled stakeholders to monitor sales performance, identify high-revenue and high-profit categories, and analyze trends over time for better strategic planning.
